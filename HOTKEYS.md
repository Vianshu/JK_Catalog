# Catalog App Hotkeys

## Navigation
Use **Alt + Key** to navigate menus. The hotkey letter will highlight in Blue when **Alt** is held.

### Main Menu
- **Alt + C**: Catalog
- **Alt + V**: Vat Working
- **Alt + T**: Tally Internal
- **Alt + S**: Settings

### Catalog Menu
- **Alt + B**: Back
- **Alt + Y**: Sync Tally
- **Alt + R**: Row Data
- **Alt + M**: Super Master (<U>M</U>aster)
- **Alt + F**: Final Data
- **Alt + G**: Full Catalo<U>g</U>
- **Alt + O**: Rep<U>o</U>rts
- **Alt + P**: Price List
- **F1**: Create CRM
- **F2**: Alter CRM
